{happy}adr : API Documentation

api/city/

Request by {postcode}:

You can use following delimiters:

  • '_' : Underscore delimiter which represent "space" and "specials" characters (" ", "'", "-").
  • '|' : Pipe delimiter which represent the "OR" operator.
  • '$' : Dollard delimiter which represent the "AND" operator.

You can call API by postcode (ex: 95000).

{happy}adr will respond results that match your request limited to 12 results.

API request:

http://www.happyadr.fr.nf/api/city/95000?token=******

happyadr/address/

Request by {address}:

You can use following delimiters:

  • '_' : Underscore delimiter which represent "space" and "specials" characters (" ", "'", "-").
  • ',' : Comma delimiter which represent the separator ("AND" operator).

You can call API by {address} (ex: rue,planchat,paris ).

{happy}adr will respond results that match your request limited to 99 results.

API request:

http://www.happyadr.fr.nf/api/address/rue,planchat,paris?token=******

API response:

{
    "http_code": 200,
    "username": "user",
    "data-size": 1,
    "data": [
        {
            "search": "rue planchat",
            "city": "20e Arrondissement",
            "postcode": "75020",
            "department": "Paris"
        }
    ]
}
